Política de Privacidade – ConectaSP

Última atualização: 2025

O aplicativo ConectaSP não coleta, armazena ou compartilha dados pessoais dos usuários.

1. Dados coletados

O aplicativo não coleta nenhum dado pessoal, como:

  • Nome
  • E-mail
  • Localização
  • Imagens
  • Dados financeiros
  • Informações do dispositivo

2. Permissões

O aplicativo não solicita permissões sensíveis. Qualquer permissão utilizada serve apenas para o funcionamento básico do app.

3. Uso dos dados

Nenhuma informação dos usuários é coletada, compartilhada, armazenada, enviada para servidores externos ou usada para fins comerciais.

4. Crianças

O aplicativo não é direcionado a crianças menores de 13 anos.

5. Segurança

Mesmo sem coletar dados, mantemos boas práticas gerais para garantir o funcionamento seguro do aplicativo.
